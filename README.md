# Skeleton project including the following libraries:

-   Express Server
-   Typescript
-   [TSOA](https://github.com/lukeautry/tsoa)
-   [TypeORM](https://github.com/typeorm/typeorm)


## How to:

To run in development mode:

```
  $ npm run dev
```

To compile and run in prod mode:

```
  $ npm run prod
```

To run tests

```
  $ npm run test
```
